# Profit Lab — Точная копия оригинального сайта на React

Полное воспроизведение сайта profit-lab.pro на React + Tailwind CSS с сохранением всех оригинальных текстов, стилей и функциональности.

## 🎨 Оригинальный дизайн

- **Цвета**: #2ba546 (зеленый), #34ae15 (светло-зеленый), белый фон
- **Шрифты**: FuturaPT (заголовки), Montserrat (текст)
- **Стиль**: Минималистичный, деловой, с карточками и плавными анимациями

## 📦 Структура сайта

### Все секции из оригинала:
1. **Hero** — главная секция с призывом к действию
2. **Benefits** — 6 направлений работы
3. **SixSteps** — 6 шагов трансформации
4. **FinanceOrder** — управленческая отчетность
5. **Modules** — 6 модулей обучения
6. **Templates** — 162 шаблона (6 категорий)
7. **WhatsIncluded** — что входит в программу
8. **AboutDarya** — о Дарье Пушкарской
9. **Testimonials** — отзывы клиентов
10. **Problems** — видео-отзывы с результатами (6 клиентов)
11. **VideoTestimonials** — дополнительные видео (4 клиента)
12. **FinalCTA** — финальный призыв "Запишись на диагностику"

## 🚀 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```

## 📱 Адаптивность

Полная адаптивность под все устройства:
- Desktop: 1200px+
- Tablet: 640px - 1199px
