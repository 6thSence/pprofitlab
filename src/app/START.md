## 🐛 Если что-то не работает

### Проблема: Ошибка при установке
```bash
# Удалить node_modules и попробовать снова
rm -rf node_modules
npm install
```

### Проблема: Порт занят
```bash
# Изменить порт в vite.config.ts
server: {
  port: 3000  # Измените на свободный порт
}
```

### Проблема: Шрифты не загружаются
```
✅ Проверьте интернет-соединение
✅ Шрифты загружаются с Google Fonts и tildacdn.com
✅ Откройте DevTools → Network для проверки
```

### Проблема: Видео не открываются
```
✅ Проверьте YouTube доступность
✅ Все видео ID актуальны
✅ Откройте console для проверки ошибок
```
