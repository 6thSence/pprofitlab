# ⚡ Быстрое развертывание Profit Lab

## 🎯 Самый простой способ (3 шага)

### 1️⃣ Установите зависимости
```bash
npm install
```

### 2️⃣ Соберите проект
```bash
npm run build
```
✅ Готово! Папка `dist/` содержит готовый сайт

### 3️⃣ Загрузите на хостинг
Загрузите **всё содержимое** папки `dist/` на ваш хостинг

---

## 📦 Что находится в папке dist/

```
dist/
├── index.html          ← главный HTML файл
├── assets/            ← JS, CSS, изображения
│   ├── index-xxxxx.js
│   ├── index-xxxxx.css
│   └── [другие файлы]
└── [другие файлы]
```

---

## 🌐 Варианты хостинга

### Вариант А: Обычный хостинг (Shared Hosting)
1. Откройте FTP/cPanel/ISPManager
2. Перейдите в папку `public_html/` или `www/`
3. Загрузите всё из `dist/`
4. Скопируйте туда же файл `.htaccess` из корня проекта
5. Готово! Откройте ваш домен

### Вариант Б: Netlify (БЕСПЛАТНО и проще всего!)
1. Зайдите на https://netlify.com
2. Перетащите папку `dist/` в браузер
3. Готово! Получите ссылку типа `your-site.netlify.app`
4. Можно подключить свой домен в настройках

### Вариант В: Vercel (тоже бесплатно)
1. Зайдите на https://vercel.com
2. Нажмите "Deploy"
3. Загрузите проект
4. Vercel всё сделает автоматически

---

## ⚠️ Важно для обычного хостинга

Обязательно загрузите файл `.htaccess` в ту же папку, куда загружаете содержимое `dist/`. Это нужно для правильной работы сайта.

## 🆘 Проблемы?

**Белый экран после загрузки?**
- Проверьте, что загрузили `.htaccess`
- Откройте F12 → Console и посмотрите ошибки

**npm install не работает?**
- Проверьте версию Node.js: `node --version` (нужна 18+)
- Скачайте Node.js с https://nodejs.org

**Нужно внести изменения?**
1. Отредактируйте нужные файлы в проекте
2. Снова выполните `npm run build`
3. Загрузите обновленные файлы из `dist/`
